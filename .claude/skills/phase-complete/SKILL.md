# Phase Complete Workflow
1. Update docs/plans/ to mark completed items with resolution notes
2. Update docs/features/ with any new design docs needed
3. Check dependencies.md and open-bugs.md for discrepancies with actual codebase
4. Write all files to docs/ directory (never .claude/plans/)
5. Summarize what was updated
